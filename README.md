# Scamdace Owens Exposed

Public static website cataloguing Candace Owens’s claims about the Charlie Kirk assassination (Sept 10, 2025 – July 19, 2026) and the public evidence that dismantles them.

**Author:** MileHigh Patriot ([@America1st5280](https://x.com/America1st5280))

## Pages

| File | Purpose |
|------|---------|
| `index.html` | Homepage |
| `claims.html` | Searchable claim catalog |
| `claim.html?id=…` | Deep-dive: claim → verdict → evidence → talking points |
| `facts.html` | Public record baseline |
| `toolkit.html` | Copy-paste combat replies |
| `submit.html` | Submit claims (primary links required) |
| `about.html` | Mission / author |
| `corrections.html` | Corrections policy |

## Content

Claims live in `js/claims-data.js`. Add new objects there to expand the catalog.
