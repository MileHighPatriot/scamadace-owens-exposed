Accountability

Corrections policy

If we misstate a date, misattribute a quote, or miss a primary source, we want it fixed. Truth is the whole point of this project.

How to request a correction

  1. Post or DM @America1st5280.
  2. Include: page URL or claim ID, what is wrong, and a source link proving the correction.
  3. Quote-level disputes need the primary clip or post — not a vague recollection.

What we will correct

  • Factual errors in dates, names, charge descriptions, or quote text
  • Broken or misassigned source links
  • Missing primary Owens statements that materially change a claim’s description
  • Verdict changes if new Tier-1 evidence warrants it

What we will not treat as a “correction”

  • “Be nicer to Candace” without factual error
  • Demands to delete true descriptions of false claims
  • Anonymous military lore offered without documents
  • Anything that requires doxxing private individuals
